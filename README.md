## 📌 عن المشروع

**أمان** منصة FinTech تُضاف كطبقة ذكاء فوق أنظمة التحويل البنكي الحالية، هدفها منع عمليات الاحتيال القائمة على الهندسة الاجتماعية قبل أن تصل الأموال إلى الطرف الخطأ.

بدلاً من تحليل الاحتيال بعد الحادثة، يتدخل **محرك الثقة** في لحظة إتمام التحويل ليُقيّم المخاطر في الوقت الفعلي ويُعطي المستخدم القدرة على اتخاذ قرار مدروس.

## 🏗️ التقنيات المستخدمة

### الواجهة الأمامية
- **React 18** + **TypeScript**
- **Vite** — بيئة تطوير سريعة
- **Tailwind CSS** — تصميم متجاوب
- **Framer Motion** — حركات سلسة
- **Wouter** — توجيه خفيف الوزن
- **TanStack Query** — إدارة حالة البيانات

### الخلفية
- **Node.js** + **Express**
- **Drizzle ORM** + **PostgreSQL**
- **Pino** — تسجيل الأحداث

### العرض التقديمي
- **فيديو الـ Pitch** — رسوم متحركة بـ React + Framer Motion
- **الشرائح** — عرض تفاعلي عربي

## 🚀 تشغيل المشروع محلياً

```bash
# استنساخ المشروع
git clone https://github.com/lo92w/Aman.git
cd Aman

# تثبيت المكتبات
pnpm install

# تشغيل الخادم الخلفي
pnpm --filter @workspace/api-server run dev

# تشغيل التطبيق الأمامي (في نافذة أخرى)
pnpm --filter @workspace/aman run dev
```

## 🎯 سيناريو الاستخدام

```
1. المستخدم يبدأ تحويلاً بنكياً
2. محرك الثقة يحلل: السجل، المبلغ، الوقت، سلوك المستخدم
3. يُحدد مسار الحماية المناسب:
   ✅ آمن → تحويل فوري
   ⚠️ مشبوه → طلب موافقة المستلم
   🚨 خطر → تأجيل 24 ساعة + تنبيه
4. المستخدم يتخذ قراراً مدروساً
```
